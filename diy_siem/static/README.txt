Optional branding assets for the Flask dashboard
==================================================

Replace the default logo by dropping your own file here, using one of these names
(first match wins):

  logo.png
  logo.svg
  logo.webp

Recommended height: ~48–64 px (width auto). Restart `python -m diy_siem serve` after
adding or changing the file.

The product title and subtitle come from .env:
  SIEM_BRAND_NAME
  SIEM_BRAND_TAGLINE
